Class: Screen

HYPER.Screen

new Screen(e)

Create a new screen. Screens and states are necessary to render and update game code.
Parameters:
Name Type Description
e object Object to pass all screen info.
Properties
Name Type Default Description
canvas string Id of the canvas you want used as the game screen.
antiAliasing boolean false Whether or not the canvas should use canvas antiAliasing.
autoClear boolean false Whether or not the canvas should clear itself before every frame.
smartRendering boolean false Whether or not to render objects not shown on the screen.
motionBlur boolean false Whether or not motion blur is enabled.
showFPS boolean false Whether or not to show the current FPS.
render function Custom rendering directions for the screen.
update function Custom updating directions for the screen.
currentState HYPER.State The current state to be used.
onClick function What happens when someone clicks.
onUp function What happens when someone clicks up.
onDown function What happens when someone clicks down.
onDblClick function What happens when someone double clicks.
onKeyUp function What happens when a key is lifted.
onKeyDown function What happens when a key is pressed.
onKeyHeld function What happens when a key is held.
fps number fps the game will be rendered and updated at.
camera object The camera object, change the position and view to change what is shown.
Properties
Name Type Default Description
x number 0 X location of the camera.
y number 0 Y location of the camera.
width number 1280 Width of the camera.
height number 720 Height of the camera.
view object The view object, change the position and view of the screen on the webpage.

Name Type Default Description
